# Yved MVP

Обновлённая версия MVP социальной сети Yved.

## Что добавлено

- исправление растягивания текста, картинок и полей ввода;
- безопасные переносы базы через ALTER TABLE без удаления старых данных;
- dev-панель с настройками сайта без деплоя кода;
- секретная кнопка в настройках: нажать маленькую точку 10 раз;
- лого Yved и favicon;
- автоплей коротких видео: активное играет, остальные ставятся на паузу;
- скрыта правая полоса прокрутки в видео-ленте;
- переходы в профили по аватаркам/авторам;
- расширенный профиль: аватар, обложка, цвет профиля;
- настройки цветов интерфейса;
- простые звуки кликов и анимации;
- базовая текстовая модерация 18+ слов;
- скрытие постов/видео через dev-панель вместо полного удаления;
- пагинация постов, видео и сообщений.

## Запуск локально

### Server

```bash
cd server
npm install
copy .env.example .env
node server.js
```

### Client

```bash
cd client
npm install
npm run dev
```

## Deploy

1. Сделай `git add . && git commit -m "update yved" && git push`.
2. Render сам обновит backend, если подключён к GitHub.
3. Vercel обнови командой:

```bash
cd client
vercel.cmd --prod
```

## Важно

Не загружай в GitHub:

- `server/.env`
- `server/database.sqlite`
- `server/uploads` с реальными файлами
- `node_modules`
